# 웹프로그램으로 데이터베이스 연동 처리
1. 지금까지 배워온 database 처리 모듈과 웹 기반 요청/모델/뷰단 템플릿을 상호 연동하여, 데이터
    베이스 처리를 웹화면에서 처리하는 것을 하고자 합니다.
2. 환경 구성
    1) front 템플릿 : 부트스트렙 환경 설정(css/jquery)
    2) backend : dao/dto/controller단 연동 처리..
3. 예제 코드를 통한 처리
    1) 사원정보 관리
        - 키워드 검색 리스트
        - 등록 처리
        - 상세 내용 확인
        - 수정 처리
        - 삭제 처리
    2) 주요 파일
        - controller : a01_empController.py
        - dao : a02_empDao.py
        - dto : z01_dto/a02_empDto.py
        - view : templates/a03_emp/a01_empList.html
                 templates/a03_emp/a02_empInsert.html
                 templates/a03_emp/a03_empDetail.html
4. 개발 순서
    1) 부트스트랩 환경 설정
    2) 초기 메인 화면 호출
    3) DB처리 : sql 확인/dto 설정/dao 메서드 호출/controller 호출
    4) controller : 요청값 처리 및 모델 처리
    5) view단 : a01_empList.html 호출..
