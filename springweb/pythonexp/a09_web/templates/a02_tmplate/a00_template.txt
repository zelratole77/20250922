# jinja2 : 파이썬용 웹 템플릿 엔진
html  안에 변수, 반복문, 조건문 등을 사용하여 동적인 웹페이지를 만들 때 사용합니다.
일반적으로 flask, django, ansible 등에서 사용됩니다.

1. 기본 기호(구분자)
    jinja2가 이건 일반 텍스트가 아니라 코드다라고 인식하는 3가지 약속입니다.
    {# ... #} : 주석(화면에 안나옴, 메모용)
    {{ ... }} : 변수 출력(화면에 값을 찍음
    {% ... %} : 제어문(if, for, set 등 로직 처리
2. 변수 출력 및 사칙연산
    변수를 화면에 찍거나, 간단한 계산을 수행합니다.
    #}
    {% set name = "홍길동" %}
    {% set count = 20 %}
    <h2>선언된 변수 호출</h2>
    <h3>이름 : {{name}}</h3>
    <h3>갯수 : {{count}}</h3>
    <h3>덧셈(+5) : {{count+5}}</h3>
    <h3>비교(count>5) : {{count>5}}</h3>

# jinja2 템플릿 엔진에서 파이프(|)기호 사용
 파이프(|) 기호는 필터를 적용할 때 사용합니다. 좌측의 변수값을 우측의 필터 함수로 전달하여 가공하는 역할을 합니다.
 1. 숫자 처리 및 형변화
    숫자를 다루거나 계산할 때 주로 사용됩니다.
    int (정수변환) : 소숫점을 버리고  정수로 변환
        {{76.5|int}} ==> 75
    float (실수변환) : 숫자를 실수형으로 변환
        {{70|float}} ==> 70.0
    round (반올림) : 지정한 소수점 자리수까지 반올림
        {{3.14159|round(2)}} ==> 3.14
        {{3.6|round(1}} ==> 4.0
    abs(절대값):음수를 양수로 변환
        {{-10:abs}} ==> 10
 2. 문자열 가공
    텍스트 데이터를 화면에 출력하기 좋게 다듬을 때 유용합니다.
    1) upper/lower  대소문자 변환
        {{'hello'|upper}} ==> HELLO
        {'WORDS'|lower} ==> words
    2) trim 공백제거
        {{'   hello  '|trim}} ==> hello  (공백제거된 문자열)
    3) truncate 문자열 자르기 : 긴 문자을 지정된 길이로 자르고 줄입표(..)표시
        {{ "동해물과 백두산이 마르고 닯도록"|truncate(10)}} ==> 동해물과 백두산이...
    4) replace 문자열 치환 : 특정 문자를 다른 문자로 변경
        {{'hello world'|replace('world','korea'}} ==> hello korea
 3. 리스트 및 컬렉션 처리
    데이터 목록을 다룰 때 사용합니다.
    1) join 문자열 합치기 : 리스트 안의 요소를 특정 구분자로 연결
       {{['사과','배','포도'] | join(',')}} ==> 사과, 배, 포도
    2) length 길이 반환 : 리스트의 개수나 문자열의 길이를 반환
        {{['a','b','c'] | length}} ==> 3
    3) sum 합계 : 리스트 내 숫자들의 합계(성적 총점 계산 시 유용
        {{[70, 80,90]|sum}} ==> 240
 4. 데이터 안전 및 기본값 설정
    default : 변수가 정의되지 않거나 값이 없을 때 대체값 출력
    {{ my_score | default(0)}}  ==> my_score가 없으면 0 출력
    safe (html 태그 허용) : 문자열에 포함된 html 태그를 텍스트가 아닌 코드로 실행
        {{'<br>굵게</br>'|safe}}
 # 숫자형 포맷
 jinja2는 파이썬의 문자열 포매팅 문법을 그대로 사용합니다. 주로 format 필터를 사용하여,
 순서가 "서식" | format(값) 형태임을 주의해야 합니다.
 %.nf  소숫점 n자리까지 표시(반올림)    {{"%.2f" | format(12.3456)}}
 %d    정수만 표시(버림 아님, 서식상 정수)   {{"%d" | format(3422.2222)}}
 %0nd  n자리수로 맞추고 빈 곳을 0으로 채우   {{'%05d'|format(7) }}
 {:,}  3자리마다 콤마(,)찍기         {{"{:,}".format(10000000)}}
