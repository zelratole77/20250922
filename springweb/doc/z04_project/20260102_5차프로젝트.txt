
## [프로젝트 개요] Spring Boot 기반의 반응형 웹 서비스 구현

본 프로젝트는 **Java Spring Boot** 백엔드 기술과 **Bootstrap** 프론트엔드 템플릿을 결합하여, 실무 중심의 웹 애플리케이션을 개발하는 것을 목표로 합니다. MVC 패턴의 명확한 이해와 데이터베이스 연동, 그리고 비동기 통신(AJAX) 및 API 활용 능력을 배양합니다.

### 1. 주요 기술 스택 (Tech Stack)

* **Backend:** Java 17, Spring Boot, Spring Data JPA/MyBatis
* **Frontend:** Bootstrap 5, JSP, jQuery
* **Database:** ORACLE
* **Communication:** AJAX (JSON 데이터 처리)
* **Special Features:** Chart.js (통계 시각화), File I/O (첨부파일 업로드/다운로드), Session 기반 인증

### 3. 주요 구현 포인트

* **데이터 모델링 (ERD):** 효율적인 데이터 저장을 위해 정규화된 DB 설계 진행.
* **비동기 처리 (AJAX):** 페이지 새로고침 없이 데이터를 갱신하여 사용자 경험(UX) 향상.
* **데이터 시각화 (Chart):** DB 내 통계 데이터를 차트로 변환하여 직관적인 대시보드 구현.
* **품질 관리:** 매주 금요일 진행되는 **코드 리뷰**를 통해 코드 컨벤션을 점검하고, 단위 테스트를 수행하여 기능의 결함을 최소화함.

* **단위 테스트:** 조원들과 함께 테스트 시나리오 만들고 각 Controller 및 Service 계층의 로직 검증.
