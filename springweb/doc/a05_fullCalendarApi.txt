# 환경 설정 및 기본 랜더링 처리
1. webapp에 다운로드한 압축파일을 풀어서 저장한다.
	https://github.com/fullcalendar/fullcalendar/releases
	fullcalendar-6.1.20.zip 다운로드
	
2. webapp\fullcalendar-6.1.20\examples\selectable.html
	확인 후, 웹서버를 통해서 로딩해본다.
	http://localhost:5050/fullcalendar-6.1.20/examples/selectable.html
	
3. controller/jsp를 통해서 초기화면 로딩
	1) 필요로 하는 lib를 공통 위치에 복사해서 설정.
		webapp\fullcalendar-6.1.20\dist
		폴드 복사해서 
	
	
		resources/static(공통img,js,css를 설정하는 위치) 하위에 복사.
	2) controller
		(java\web\com\springweb\a08_fullcalendar\A01_FullCalenderController.java)
		/calendarList
		
		view : webapp\WEB-INF\views\a08_ajax\a09_calendar.jsp
	3) a09_calendar.jsp
		공통 lib js를 설정
		selectable.html 파일에서 script 이후 부분을 복사
		
		http://localhost:5050/calendarList 을 통해서 최종 확인..
