> 요청(Request) 흐름:
> 클라이언트(JSON 문자열/Form Data) → Controller( (A) 객체로 매핑 ) → Service → DAO → DB(Table Row)
> 응답(Response) 흐름:
> DB → DAO( (A) 객체 ) → Service(성공/실패 메시지 문자열) → Controller( (B) 객체 ) → 클라이언트(AJAX success 콜백)
