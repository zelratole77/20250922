## Ⅰ. 객관식 (사지선다형) - 20문항

1. **스프링 MVC 패턴에서 사용자의 요청(URL)을 가장 먼저 받아 처리하는 구성 요소는?**
① View
② Service
③ Controller
④ Dao
2. **`@RequestMapping("idolGrpList")` 어노테이션의 주요 역할로 올바른 것은?**
① 데이터베이스 테이블과 클래스를 매핑한다.
② 특정 URL 요청을 컨트롤러의 메서드와 연결한다.
③ 비즈니스 로직을 수행하도록 서비스 객체를 생성한다.
④ 자바 객체를 JSON 형태로 변환한다.
3. **`A03_Idol_Dao` 인터페이스에 `@Mapper` 어노테이션을 사용하는 주된 이유는?**
① Mybatis가 해당 인터페이스를 구현한 프록시 객체를 생성하게 하기 위해
② 해당 클래스가 뷰(JSP)임을 명시하기 위해
③ 브라우저의 캐시 메모리를 초기화하기 위해
④ 서버의 포트 번호를 설정하기 위해
4. **Service 단에서 `sch.setGroupName("%"+sch.getGroupName()+"%")` 처리를 하는 목적은?**
① 데이터를 암호화하여 보안을 강화하기 위함
② 데이터베이스에서 SQL의 `LIKE` 연산자를 이용한 부분 일치 검색을 하기 위함
③ NULL 포인터 예외를 방지하기 위함
④ 데이터의 길이를 제한하기 위함
5. **`A01_Idol_Controller`에서 `Model d` 파라미터를 사용하는 이유는?**

④ 서버의 환경 설정값을 가져오기 위해
6. **조회 리스트 화면(`a01_groupList.jsp`)에서 검색 조건이 없을 때 전체 리스트가 나오게 하려면 Service에서 어떤 처리가 필요한가?**
① 요청값이 null일 때 프로그램을 종료시킨다.
② 요청값이 null일 때 빈 문자열("")로 치환하고 앞뒤에 "%"를 붙인다.
③ SQL 문에서 WHERE 절을 삭제한다.
④ Controller에서 에러 페이지로 리다이렉트한다.
7. **Mybatis의 `@Insert` 어노테이션 내 SQL에서 자바 객체의 필드값을 매핑할 때 사용하는 문법은?**
① `${field}`
② `#{field}`
③ `&{field}`
④ `*{field}`

10. **JSP에서 `<c:forEach var="grp" items="${grpList}">` 태그를 사용할 때 `items` 속성에 들어가는 값의 출처는?**
