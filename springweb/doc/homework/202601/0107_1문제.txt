
[일일정리문제]
## 1. 환경 설정 및 기본 렌더링

1. [개념] FullCalendar 6버전 이상을 스프링 프로젝트에 통합할 때, `dist` 폴더 내의 정적 파일(js, css)들을 주로 어느 경로에 배치하며 JSP에서 어떤 방식으로 참조해야 합니까?
2. [실습] 여러분은 지금 막 새로운 프로젝트에 투입되었습니다. 팀장님이 "공식 홈페이지의 `selectable.html` 예제와 똑같은 화면을 우리 스프링 프로젝트의 `/calendar` 경로에서 보고 싶다"고 요청했습니다. 외부 CDN을 사용하지 않고, 다운로드한 라이브러리 파일을 `webapp\WEB-INF\views\a08_ajax\`에 넣어 프로젝트를 구동해 보세요.

## 2. JSON 데이터와 속성 이해

3. [개념] FullCalendar의 `events` 속성에서 `start`와 `end` 날짜를 지정할 때, 시간 정보(T09:00:00)가 포함된 데이터와 포함되지 않은 데이터는 화면상에서 어떤 차이를 보입니까?
4. [실습] "코딩 공부하기"라는 일정을 만드는데, 이 일정은 2026년 1월 7일 오전 9시에 시작해서 오후 6시에 끝나야 하며, 클릭했을 때 내 개인 블로그로 이동해야 합니다. 이 요구사항을 충족하는 JSON 객체를 작성해 보세요.

## 3. DB 설계 및 DTO 매핑

5. [개념] 데이터베이스의 `ALL_DAY` 컬럼은 보통 숫자형(0 또는 1)으로 설계하는데, 자바의 DTO(Calendar.java)에서 이를 `boolean` 타입으로 선언하는 이유는 무엇입니까?
6. [실습] 회의실 예약 시스템(`calendar_room`)을 위해 테이블을 설계해 보세요. 기존 일정 테이블 속성에 추가로 '예약할 회의실 번호(room_no)'와 '참석 인원(headcount)' 컬럼을 포함하고, 이에 대응하는 자바 VO 클래스를 작성해 보세요.

## 4. 일정 조회 처리 (Back-end)

7. [개념] `@ResponseBody` 또는 `ResponseEntity`를 사용하여 리스트 데이터를 반환할 때, SQL의 컬럼명(`START1`)과 DTO의 필드명(`start`)이 다를 경우 MyBatis에서 이를 해결하는 가장 일반적인 방법은 무엇입니까?
8. [실습] 회의실 예약 현황을 불러오는 `calendarRoomList` 기능을 구현하세요. 현재 날짜로부터 일주일 이내의 예약 데이터만 조회해오는 SQL 쿼리를 작성하고, 이를 Service단에서 호출하는 로직을 구성해 보세요.

## 5. AJAX를 이용한 프런트엔드 연동

9. [개념] FullCalendar의 `events` 속성을 배열(`[]`)이 아닌 함수(`function(info, successCallback, ... )`) 형태로 작성했을 때 얻을 수 있는 장점은 무엇입니까?
10. [실습] 서버에서 받아온 데이터 중 `writer`가 "관리자"인 경우에만 배경색을 빨간색으로 변경하여 화면에 뿌려주고 싶습니다. `successCallback`을 호출하기 전, 자바스크립트의 `map()` 함수를 활용하여 데이터를 가공하는 코드를 작성해 보세요.

## 6. 일정 추가 및 상세 이벤트 처리

11. [개념] 사용자가 캘린더의 빈 날짜를 드래그하거나 클릭했을 때 발생하는 이벤트 이름은 무엇이며, 이때 전달받는 `arg` 매개변수에서 시작 날짜와 끝 날짜 문자열을 추출하는 속성명은 무엇입니까?
12. [실습] 사용자가 날짜를 클릭하면 브라우저의 기본 `prompt` 창이 아닌, 예쁜 모달(Modal) 창이 뜨면서 일정 제목과 내용을 입력받는 구조를 상상해 보세요. 모달에서 입력받은 데이터를 `calendar.addEvent()` 함수를 사용해 화면에 즉시 반영하는 코드를 작성해 보세요.
