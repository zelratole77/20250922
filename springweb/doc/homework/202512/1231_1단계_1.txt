## 1. [개념 정리] 주관식 문제 (7문제)

**Q1. [서버 경로 설정]**
`application.properties`에 설정된 외부 경로(예: `user.upload`)를 스프링 컨트롤러에서 가져와 사용하기 위해 필드 변수 위에 선언하는 어노테이션은 무엇인가요?

**Q2. [파일 객체 변환]**
파일의 실제 경로를 다루기 위해 `Paths.get(path)`를 통해 생성된 `Path` 객체를, 스프링에서 리소스로 활용하기 위해 최종적으로 변환하는 클래스의 명칭은 무엇인가요?

**Q3. [파일 다운로드 헤더]**
브라우저가 파일을 열지 않고 파일 저장 대화상자를 띄우도록 설정할 때, `HttpHeaders.CONTENT_DISPOSITION` 속성에 할당해야 하는 필수 값 형식은 무엇인가요? (파일명 포함 형식)

**Q4. [네트워크 인코딩]**
한글 파일명이 포함된 URL을 전송할 때, 공백이나 특수문자가 깨지지 않도록 하기 위해 `URLEncoder`를 사용하여 처리하는 표준 인코딩 방식(Charset)은 무엇인가요?

**Q5. [AJAX 속성]**
jQuery AJAX에서 요청이 성공적으로 완료되었을 때 실행될 콜백 함수를 지정하는 속성명은 무엇인가요?

**Q6. [MIME 타입 처리]**
파일의 확장자에 따라 `image/png`, `text/plain` 등을 자동으로 식별해주는 Java의 메서드는 `Files.__________` 입니다. 빈칸에 들어갈 메서드명은 무엇인가요?

**Q7. [비동기 응답 처리]**
AJAX 요청 시 컨트롤러 메서드가 JSP 뷰를 찾지 않고 데이터(문자열, JSON 등) 자체를 직접 브라우저에 전송하게 만드는 핵심 어노테이션은 무엇인가요?

## 2. [실습 응용] 문제 (5문제)

> **상황:** 프로젝트 관리 시스템에서 '보고서 파일' 목록을 조회하고 다운로드하는 기능을 구현해야 합니다.

**실습 1: [Controller 매핑]**
URL `/reportDownload`로 요청이 들어오면 `reportName`이라는 파라미터를 받아 파일을 리턴하는 메서드 선언부를 작성하세요. (리턴 타입은 `ResponseEntity<Resource>` 사용)

**실습 2: [AJAX 실습]**
서버의 `/checkServerTime`이라는 경로를 AJAX로 호출하여 서버에서 보내주는 현재 시간 메시지를 콘솔(`console.log`)에 출력하는 코드를 작성하세요.

**실습 3: [파일 존재 여부 확인]**
다운로드 컨트롤러 내에서 `Resource` 객체가 실제 물리적 경로에 존재하는지 체크하여, 없을 경우 `ResponseEntity.notFound()`를 리턴하는 조건문을 작성하세요.

**실습 4: [JSP 다운로드 이벤트]**
파일 목록 테이블에서 특정 행을 **더블 클릭(ondblclick)** 했을 때, 파일명(예: 'final_report.pdf')을 인자로 받아 `/reportDownload?reportName=파일명`으로 이동시키는 자바스크립트 함수 `goDownload(fname)`을 작성하세요.

**실습 5: [MIME 타입 예외 처리]**
파일의 타입을 알 수 없을 때(null일 때) `contentType`을 모든 이진 파일을 의미하는 `application/octet-stream`으로 설정하는 삼항 연산자 또는 if문을 작성하세요.
